# MEMORA

memora es una aplicación web de comercio electrónico para funerarias, diseñada para ofrecer una experiencia segura y empática a los clientes. Permite visualizar de manera clara un catálogo digital de ataúdes y servicios adicionales, así como seleccionar, contratar y realizar el pago completo o parcial de los servicios de forma remota.

Características

Catálogo digital de ataúdes y servicios funerarios
Visualización clara de precios y opciones
Selección y personalización de servicios
Contratación y pago seguro (completo o parcial) de manera remota
Transparencia y agilidad en el proceso de contratación
Interfaz empática y fácil de usar
